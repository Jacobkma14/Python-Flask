🚀 Project: ISS Tracker – Real-Time Satellite Location, Weather & Geospatial Insights with Flask

🔍 Project Description

A Flask-based web app that tracks the International Space Station (ISS) in real-time. It combines multiple APIs to provide:

  • 📍 Live latitude and longitude of the ISS
  • 🌤️ Real-time weather data for the ISS location
  • 🗺️ Reverse geolocation with country name and flag
  • 📏 Distance from Sudbury, Ontario

Tools Used: Flask, Open Notify API, OpenWeatherMap API, REST Countries API, Python

🧠 ISS Tracker – Logic Flow

User opens the app
↓
App requests ISS location
↓
App gets weather at location
↓
App calculates distance from your location
↓
App reverse geocodes country
↓
If over land:
Show country name & flag

If over water:
Show "The ISS is over water"
↓
Display all data on webpage
